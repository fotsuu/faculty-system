# CRMS Reports Guide

## Overview
The CRMS provides comprehensive reporting capabilities for faculty members to generate, view, and manage grade reports, attendance records, and performance analytics.

---

## Available Reports

### 1. 📊 Student Grade Report
**Purpose:** Detailed record of all student grades for your subjects

**What it includes:**
- Subject Code and Name
- Student ID and Full Name
- Program and Year Level
- Grade Point (numerical grade on 1.0-5.0 scale)
- Letter Grade (A, B, C, D, F)
- Raw Grade (Special grades: INC, DR, W, P)
- Date Recorded
- Semester and School Year
- Class Section

**Ideal for:**
- Official Report of Grades (ROG) submission
- Academic records and documentation
- Grade verification and audit trails
- Student transcript preparation

**Format:** CSV (Excel compatible)

---

### 2. 📈 Pass/Fail Analysis Report
**Purpose:** Summary of student achievement and performance across subjects

**What it includes:**
- Subject Code and Name
- Total Students in Subject
- Number of Passing Students
- Number of Failing Students
- Pass Rate (%) - Calculated as (Passed / Total) × 100

**Grading Scale:**
- **Pass:** Grade Point ≤ 3.0 (A, B, C, D)
- **Fail:** Grade Point > 3.0 (F)

**Ideal for:**
- Performance analysis by subject
- Identifying struggling subjects
- Course effectiveness evaluation
- Statistical reporting to administration

**Format:** CSV with summary statistics

---

### 3. 📋 Attendance Summary Report
**Purpose:** Complete tracking of student attendance patterns

**What it includes:**
- Subject Code and Name
- Total Sessions Conducted
- Present Count
- Late Count
- Absent Count
- Excused Count
- Attendance Percentage

**Attendance Scoring:**
- Present: 1.0 point
- Late: 0.5 point
- Absent: 0 points
- Excused: 0 points

**Attendance Percentage Calculation:**
- (Present Points + Late Points) / Total Possible Points × 100

**Ideal for:**
- Monitoring student attendance
- Identifying chronic absenteeism
- Documentation for academic counseling
- Compliance with institutional attendance policies

**Format:** CSV with attendance statistics

---

### 4. 🎓 Lecture & Laboratory Summary Report
**Purpose:** Performance breakdown comparing lecture and laboratory components

**What it includes:**
- Subject Code and Name
- Lab Performance (%) - 60% weight
- Lecture Performance (%) - 40% weight
- Overall Score (%) - Combined weighted average

**Calculation Method:**
- Lab Performance: Average grade converted to percentage (60% contribution)
- Lecture Performance: Average grade converted to percentage (40% contribution)
- Overall Score: Lab Performance × 0.6 + Lecture Performance × 0.4

**Ideal for:**
- Identifying strengths in lab vs. lecture components
- Curriculum evaluation
- Assessing teaching effectiveness
- Student support planning (tutoring/remediation)

**Format:** CSV with performance breakdown

---

### 5. 📑 Comprehensive Analytics Report
**Purpose:** All-in-one report combining all available analytics

**What it includes:**
- Complete Grade Distribution Analysis
- Pass/Fail Statistics Across All Subjects
- Attendance Trends and Patterns
- Quiz Performance Analytics
- Exam Performance Summary
- Student Performance Rankings
- Subject Summary Statistics

**Ideal for:**
- Complete academic overview
- End-of-semester comprehensive analysis
- Program evaluation meetings
- Documentation for accreditation reviews

**Format:** Structured CSV with multiple sections

## How to Generate Reports

### Step 1: Access the Report Generator
1. Go to your Faculty Dashboard
2. Scroll to the **"Grade Reports & Analytics"** section
3. You will see the report generator form

### Step 2: Select Report Type
- **Student Grade Report** - For individual student grades
- **Pass/Fail Analysis** - For performance statistics
- **Attendance Summary** - For attendance tracking
- **Lecture & Lab Summary** - For component breakdown
- **Comprehensive Analytics** - For complete report

### Step 3: (Optional) Select a Specific Subject
- Leave blank to include **All Subjects**
- Or choose a specific subject from the dropdown

### Step 4: Generate
- Click the **"Generate Report"** button
- The system will process and create your report
- You'll be redirected to the Reports page

---

## Managing Your Reports

### View Reports
1. Go to **"Your Generated Reports"** section on Dashboard
2. Or visit the **Reports** page from the sidebar
3. Click **"View"** to see report in browser

### Download Reports
1. Click **"Download"** button next to any report
2. Report downloads as CSV file
3. Open with Excel, Google Sheets, or any spreadsheet application

### Advanced Features
- **Print Reports:** Click "Print Report" when viewing report
- **Search Reports:** Use the search bar on Reports page
- **Sort Reports:** Reports list sorted by most recent first

## Understanding the Grading Scale

The grading scale used in CRMS is based on Philippine standards:

| Letter Grade | Numeric Range | Interpretation |
|-------------|---------------|-----------------|
| **A** | 1.0 - 1.5 | Excellent |
| **B** | 1.51 - 2.0 | Good |
| **C** | 2.01 - 2.5 | Fair |
| **D** | 2.51 - 3.0 | Passing |
| **F** | 3.01 - 5.0 | Failing |
| **INC** | — | Incomplete |
| **DR** | — | Dropped |
| **W** | — | Withdrawn |
| **P** | — | Passed (No grade) |

## Report Generation Settings

### Subject-Specific Reports
Select a subject to generate reports for that subject only. This is useful for:
- Single subject analysis
- Focused remediation planning
- Detailed subject performance review

### All Subjects Reports
Leave subject blank to include data from all your subjects:
- Comprehensive analysis across all courses
- Workload and performance overview
- Comparative analysis between subjects

## Best Practices

### Regular Reporting
- Generate reports weekly to track progress
- Generate monthly summaries for planning purposes
- Generate semester-end comprehensive reports

### Data Accuracy
- Verify grades are entered correctly before generating reports
- Check attendance records regularly
- Review reports for anomalies or discrepancies

### Report Filing
- Download and archive important reports
- Keep copies for documentation
- Submit required reports within institutional deadlines

### Using Reports for Improvement
- Identify underperforming students early
- Adjust teaching methods based on performance data
- Plan interventions for struggling students
- Celebrate high achievement and effective teaching

## Troubleshooting

### No Data Appears in Report
- Ensure you have uploaded grade data
- Check that students are enrolled in subjects
- Verify attendance records have been entered

### Missing Students in Report
- Confirm students are linked to the subject
- Check if grades/attendance data exists for the student
- Verify student records in system

### Report Takes Long Time to Generate
- This is normal for large classes (100+ students)
- System processes all records and calculations
- Wait for completion before navigating away

### Report Download Issues
- Ensure your browser allows CSV downloads
- Try a different browser if having issues
- Check download folder for the file
